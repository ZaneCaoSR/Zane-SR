## 二、技术方案

### 2.1 技术栈

| 类型 | 选型 | 说明 |
|------|------|------|
| 框架 | 微信小程序原生 | 无框架依赖，体积小 |
| 后端 | 微信云开发 | 云函数 + 云数据库 + 云存储 |
| AI 服务 | 阿里云视觉智能 | 婴儿表情识别 + 图像描述 |
| 图片压缩 | Canvas 2D | 小程序内置，无额外依赖 |

### 2.2 核心 API

| 功能 | API | 页面 |
|------|-----|------|
| 拍照 | `wx.chooseMedia({ mediaType: ['image'], sourceType: ['camera'] })` | upload |
| 相册选择 | `wx.chooseMedia({ mediaType: ['image'], sourceType: ['album'], count: 9 })` | upload |
| 图片压缩 | `wx.compressImage()` + Canvas | upload |
| 云存储上传 | `wx.cloud.uploadFile()` | upload |
| 数据库查询 | `db.collection('photos').where()` | album |
| 云函数调用 | `wx.cloud.callFunction()` | photo |

### 2.3 数据库设计 (MVP)

```javascript
// photos 表 - 照片存储
{
  _id: ObjectId,
  fileID: String,           // 云存储文件ID
  babyId: ObjectId,         // 关联宝宝ID
  uploadTime: Date,         // 上传时间
  shotDate: Date,           // 拍摄日期（可自定义）
  month: Number,            // 所属月份（用于分组）
  year: Number,             // 所属年份
  
  // AI 识别结果
  aiResult: {
    expressions: [String], // ['smile', 'laugh']
    actions: [String],      // ['raise_head', 'roll_over']
    description: String,   // AI 生成的描述
    confidence: Number     // 识别置信度
  },
  
  // 用户数据
  remark: String,           // 用户备注
  isMilestone: Boolean,    // 是否为里程碑
  
  _createTime: Date,
  _updateTime: Date
}

// baby_info 表 - 宝宝信息
{
  _id: ObjectId,
  name: String,             // 宝宝名字
  birthDate: Date,         // 生日
  gender: String,           // 性别
  privacy: {
    allowShare: Boolean,   // 允许分享
    allowAI: Boolean       // 允许AI识别
  },
  _createTime: Date,
  _updateTime: Date
}
```

### 2.4 云函数设计

| 云函数 | 功能 |
|--------|------|
| `ai-analyze` | 调用 AI 服务识别图片 |
| `get-month-photos` | 按月份获取照片列表 |
| `update-photo` | 更新照片备注/标签 |
| `delete-photo` | 删除照片及云存储文件 |
| `get-baby-info` | 获取宝宝信息 |
| `save-baby-info` | 保存/更新宝宝信息 |

## 三、页面详细设计

### 3.1 相册首页 (T1)

**入口**: `/pages/album/index`

| 模块 | 说明 |
|------|------|
| 顶部导航栏 | 标题"宝宝相册"，右侧设置按钮 |
| 月份时间线 | 横向滚动月份卡片，点击展开当月照片 |
| 里程碑入口 | 标记重要瞬间的快捷入口 |
| 拍照快捷按钮 | 右下角悬浮 FAB 按钮，点击跳转上传页 |
| 空状态 | 无照片时显示引导上传提示 |

**数据结构**:
```javascript
// 按月份分组的数据
[
  {
    year: 2026,
    month: 3,
    monthLabel: '2026年3月',
    coverPhoto: 'cloud://xxx',  // 封面图
    count: 12,                  // 照片数
    photos: [/* 照片列表 */]
  }
]
```

### 3.2 上传页面 (T2 + T3)

**入口**: `/pages/upload/index`

| 模块 | 说明 |
|------|------|
| 拍照按钮 | 打开相机拍照 |
| 相册选择 | 打开相册多选（最多9张）|
| 压缩预览 | 压缩后预览效果 |
| 上传进度 | 显示上传和 AI 识别进度 |
| AI 识别状态 | 识别中/识别完成/识别失败 |

**流程**:
```
选择照片 → 压缩处理 → 上传到云存储 → 调用 AI 识别 → 保存到数据库 → 跳转详情页
```

### 3.3 照片详情页 (T6)

**入口**: `/pages/photo/index?id=xxx`

| 模块 | 说明 |
|------|------|
| 大图展示 | 全屏查看照片 |
| AI 描述 | 显示 AI 生成的描述文本 |
| 表情标签 | 展示识别出的表情 |
| 动作标签 | 展示识别出的动作 |
| 备注编辑 | 可编辑的备注字段 |
| 删除按钮 | 删除照片 |
| 返回相册 | 返回首页 |

### 3.4 宝宝信息设置 (T7)

**入口**: `/pages/settings/index`

| 模块 | 说明 |
|------|------|
| 宝宝头像 | 头像区域（可上传）|
| 宝宝名字 | 输入框 |
| 宝宝生日 | 日期选择器 |
| 隐私设置 | 分享权限、AI 识别开关 |
| 保存按钮 | 保存设置到数据库 |

## 五、注意事项

### 页面功能说明

1. **相册首页** (`pages/album/`)
   - 月份时间线横向滚动
   - 照片网格展示
   - 里程碑标记显示
   - 悬浮拍照/相册上传按钮

2. **上传页面** (`pages/upload/`)
   - 支持拍照和相册选择
   - 图片自动压缩
   - 上传进度显示
   - AI 识别进度展示

3. **照片详情页** (`pages/photo/`)
   - 大图预览
   - AI 识别结果展示（表情、动作）
   - 备注编辑功能
   - 里程碑标记/取消
   - 删除照片（含确认弹窗）

4. **设置页面** (`pages/settings/`)
   - 宝宝头像上传
   - 宝宝名字、生日、性别
   - 隐私设置（AI识别、分享开关）
